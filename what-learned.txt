1. we need to install firebase on react project for connect firebase to react app.
    ==> npm install --save firebase / npm install firebase --save

2. Then we need to import everything(*) as firebase from 'firebase/app'
    ==> import * as firebase from 'firebase/app'

3. Then we need to make a firebaseConfig.js file on the project main folder. 

4. Then we need to copy the firebase config obj from project settings option of firebase project overview menu.
