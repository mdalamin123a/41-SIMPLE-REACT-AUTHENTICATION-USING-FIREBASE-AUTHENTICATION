1. signInWithPopup() method is used to sign in user with google with popup. the 
first arg is getAuth() returns vlaue, 2nd arg is provider. And .then() is used 
to recieve the response and the arg is a callback function. and the parameter 
of the callback function is result.
    ==> import {getAuth, GoogleAuthProvider, signInWithPopup} from 'firebase/auth';
        firebase.initializeApp(firebaseConf);
        function App(){
            const provider = GoogleAuthProvider();
            const handleSignin = () => {
                signInWithPopup(getAuth(), provider)
                .then((result) => {
                    const {displayName, email, photoURL} = result.user;
                    console.log(displayName, email, photoURL);
                });
            }
        }
